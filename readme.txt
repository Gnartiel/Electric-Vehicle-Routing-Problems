Có 5 thư mục:
* Thư mục Codes chứa:
	+ Code thực nghiệm gồm các file .cpp, .hpp, .exe, .o
	+ File .ipynb để visualize kết quả
	+ Các file evrp để chạy
* Thư mục Docs chứa file .pdf báo cáo
* Thư mục Result chứa các kết quả chạy thực nghiệm
* Thư mục Slides chứa 2 file slide: 1 file .pdf và 1 file .pptx
* Thư mục Videos chứa link youtube của video và file video.mp4
_______________________________________________________________
Để chạy code thực nghiệm theo từng phương pháp cần: 
* uncomment 1 hàm cần chạy trong các hàm được gọi ở hàm run_heuristic() thuộc file heuristic.cpp 
* chạy lệnh `make` ở thư mục Codes
* chạy lệnh `main link_file_evrp_in_folder`
